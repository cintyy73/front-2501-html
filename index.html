Bienvenidas al Mundo del Diseño Web

Descubrí cómo transformar ideas en páginas web modernas y funcionales. En este espacio aprenderás las tecnologías más usadas para el desarrollo frontend: desde la estructura con HTML, el estilo con CSS, hasta la interactividad con React y frameworks UI como Bulma y Chakra UI. ¡Tu camino para ser desarrolladora web empieza aquí!

¿Qué vas a aprender?

Nuestro enfoque está en brindarte conocimientos prácticos y actuales para crear sitios web que se adapten a cualquier dispositivo. Vas a dominar:

  • Diseño Mobile First para priorizar la experiencia en celulares y tablets.
  • Uso de unidades relativas como %, vh, vw y rem que permiten layouts flexibles.
  • Media Queries para ajustar estilos según el tamaño de pantalla.
  • Técnicas visuales modernas usando CSS Grid y Flexbox para organizar contenidos con precisión y dinamismo.

Contenidos prácticos y dinámicos desde cero:

  • Introducción a HTML: Comprendé la estructura básica de un documento web y cómo organizar contenido semánticamente.
  • Estilos con CSS: Aprendé a diseñar con buenas prácticas, colores, tipografías y efectos que mejoran la experiencia visual.
  • Diseño responsivo: Asegurá que tus páginas se vean bien en cualquier dispositivo, desde smartphones hasta monitores grandes.
  • Mini-proyectos: Aplicá lo aprendido construyendo proyectos reales, que te ayudarán a reforzar conceptos y armar un portafolio.

Galería de Proyectos

Explorá una selección de trabajos prácticos donde aplicamos todo lo aprendido. Desde generadores de memes, buscadores de películas, hasta portfolios profesionales y pronósticos climáticos. Cada proyecto es una oportunidad para practicar diseño, funcionalidad y buena estructura de código.
