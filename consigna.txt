🎯 Objetivo general:
Construir una landing page informativa para una ONG ficticia, integrando conceptos clave de HTML y CSS trabajados a lo largo del curso.

📚 Contenidos a evaluar:
Estructura semántica en HTML

Estilos con CSS y selectores

Modelo de caja

Flexbox

SVG

Responsive design

Formularios

Selectores avanzados y pseudo-clases

📝 Consigna paso a paso
1. 🏗️ Estructura base en HTML5
Crear un archivo index.html.

Usar etiquetas semánticas: header, nav, main, section, article, footer.

Incluir un título principal (<h1>) y subtítulos (<h2>, <h3>).

2. 🎨 Estilos base (Intro CSS + parte 2)
Crear un archivo style.css enlazado correctamente.

Aplicar fuentes, colores y estilos de texto.

Usar selectores de etiquetas, clases e IDs.

Aplicar :hover, :focus, :nth-child, etc.

3. 📦 Box Model
Aplicar márgenes (margin), relleno (padding), bordes (border) y box-sizing: border-box.

Usar width, height, max-width, etc.

4. 🧱 Layout con Flexbox
Crear una galería de tarjetas (mínimo 3) con Flexbox.

Cada tarjeta debe contener:

Imagen (puede ser SVG)

Título

Descripción corta

Botón (con pseudo-clases :hover)

5. 🧩 Uso de SVG
Insertar al menos un icono SVG (puede ser inline o externo).

Puede ser parte del logo o íconos en botones.

6. 📱 Responsive Design
Adaptar el sitio a dispositivos móviles.

Usar media queries (@media) para cambiar el layout en pantallas menores a 768px.

En móviles, los elementos deben apilarse de forma vertical.

7. 📝 Formulario de contacto
Crear un formulario con los siguientes campos:

Nombre (obligatorio)

Email (obligatorio y validado)

Mensaje (textarea)

¿Cómo nos conociste? (select)

Acepta términos (checkbox)

Estilizar usando :focus, :valid, :invalid, :placeholder-shown.

8. 🎯 Selectores avanzados
Usar:

Selector de atributos (input[type="text"])

nth-child, nth-of-type

Selector de hermano (+, ~)

Selector universal (*)

Selector descendiente (nav ul li a)

9. 🧪 Accesibilidad y buenas prácticas
Usar etiquetas label asociadas a los campos del formulario.

Usar alt en las imágenes.

Verificar estructura correcta y cierre de etiquetas.
